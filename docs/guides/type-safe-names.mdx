mujoco-react uses TypeScript declaration merging for model resource names. The recommended workflow is the same shape as TanStack Router route generation: a Vite plugin scans your source files and writes a checked-in generated type artifact.

## Vite Plugin

```ts
// vite.config.ts
import { defineConfig } from "vite";
import react from "@vitejs/plugin-react";
import { mujocoReact } from "mujoco-react/vite";

export default defineConfig({
  plugins: [
    react(),
    mujocoReact({
      models: "models/panda/scene.xml",
    }),
  ],
});
```

The plugin writes `src/mujoco-register.gen.d.ts` during dev and build:

```ts
// Auto-generated by mujoco-react. Do not edit.

import "mujoco-react";

declare module "mujoco-react" {
  interface Register {
    actuators: "joint1" | "joint2" | "gripper";
    sensors: "imu_gyro";
    bodies: "link0" | "hand";
    joints: "joint1" | "joint2";
    sites: "tcp";
    geoms: "floor";
    keyframes: "home";
  }
}
```

Commit this generated file. It gives autocomplete and type checking without adding runtime cost.

```tsx
useCtrl("gripper");
useSensor("imu_gyro");
useIkController({ siteName: "tcp" });
api.applyForce("hand", force);
```

## Multiple Models

```ts
mujocoReact({
  models: [
    "models/panda/scene.xml",
    "models/spot/scene.xml",
  ],
  generatedRegister: "src/mujoco-register.gen.d.ts",
});
```

Names from all configured models are merged into the same `Register` augmentation.

## CLI Fallback

Use the CLI for non-Vite projects or one-off generation:

```bash
npx mujoco-react codegen models/panda/scene.xml
```

Legacy alias:

```bash
npx mujoco-react-codegen models/panda/scene.xml
```

## Options

| Option | Type | Default | Description |
|--------|------|---------|-------------|
| `models` | `string \| string[]` | required | Entry MJCF/URDF files to scan |
| `generatedRegister` | `string` | `src/mujoco-register.gen.d.ts` | Output declaration file |
| `moduleName` | `string` | `mujoco-react` | Module to augment |
| `disableLogging` | `boolean` | `false` | Disable plugin console output |
